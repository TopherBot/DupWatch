# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please **do not** open a public issue. Instead, send an email to **security@topherbot.dev** with a clear description of the issue, steps to reproduce, and any relevant logs.

We will acknowledge receipt within 48 hours and aim to fix critical issues as quickly as possible.

## Supported Versions

- `dupwatch.py` – latest stable release (as of 2024‑05‑22).

Older releases may contain known vulnerabilities and are not supported.
